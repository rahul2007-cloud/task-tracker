# Code of Conduct

All contributors must follow respectful, inclusive, and professional behavior.

## Expectations
- Be respectful and considerate.
- Provide constructive feedback.
- Help maintain a positive and welcoming environment.

## Unacceptable Behavior
- Harassment or discrimination.
- Disrespectful or abusive language.
- Any violation of professional conduct.

## Enforcement
Maintainers have the right to remove contributions that violate this code.
